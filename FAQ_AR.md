# أسئلة شائعة - محاسب برو

## ❓ الأسئلة الشائعة

### هل يدعم النظام لغات أخرى؟

حالياً، النظام مدعوم باللغة العربية فقط. يمكن إضافة دعم لغات أخرى في النسخ المستقبلية من خلال نظام i18n.

### كيف أضيف مستخدم جديد؟

1. سجّل الدخول كـ Admin
2. اذهب إلى إدارة المستخدمين (ستُضاف قريباً)
3. انقر على "إضافة مستخدم"
4. أدخل البيانات وحدد الدور

### هل يمكن تعديل الفاتورة بعد إنشاؤها؟

حالياً، لا يمكن تعديل الفواتير المنشورة. لكن يمكن إنشاء فاتورة دائنة (Credit Note).

### كيف أقوم بالنسخ الاحتياطية؟

```bash
# تصدير قاعدة البيانات
mysqldump -u root -p accounterp_db > backup.sql

# استيراد النسخة الاحتياطية
mysql -u root -p accounterp_db < backup.sql
```

### هل يمكن نقل البيانات من نظام آخر؟

نعم، يمكن استيراد البيانات من ملفات Excel باستخدام Scripts خاصة. تواصل معنا للمساعدة.

### ما هي متطلبات الأمان الدنيا؟

- كلمات مرور قوية (8+ أحرف، أرقام، رموز)
- تحديث النظام بانتظام
- نسخ احتياطية دورية
- استخدام HTTPS في الإنتاج
- تحديد صلاحيات صريحة للموظفين

### هل يدعم النظام التعامل مع عملات متعددة؟

حالياً لا. الإصدار الحالي يعمل بعملة واحدة. سيتم إضافة دعم العملات المتعددة قريباً.

### كيف أقوم بإصدار الفواتير الإلكترونية؟

هذه الميزة قيد التطوير وستُطلق في النسخة القادمة.

### ما هو الفرق بين الأدوار المختلفة؟

| الدور | الصلاحيات |
|------|----------|
| Admin | كل شيء |
| المحاسب | الفواتير والحسابات والتقارير |
| موظف المبيعات | فواتير المبيعات فقط |
| أمين المخزن | المخزون والمشتريات فقط |

### هل يمكن استخدام النظام Offline؟

حالياً لا. النسخة الحالية تتطلب اتصال إنترنت. يمكن تطوير نسخة offline لاحقاً.

### كيف أقوم بالنسخ الاحتياطية التلقائية؟

استخدم Cron Jobs على Linux:

```bash
# اضف هذا إلى crontab
0 2 * * * mysqldump -u root -p'password' accounterp_db > /backups/db_$(date +\%Y\%m\%d).sql
```

### هل يوجد API توثيق كاملة؟

نعم، جميع الـ APIs موثقة في ملف API.md (سيُضاف قريباً).
